Обзор Добавляет шлемы с функцией Create Mod Goggle и стандартные доспехи. Если вы хотите носить полную незеритовую броню, но также хотите видеть статистику очков или использовать свой задний танк, этот мод для вас!

Рецепты Бесформенный Ваниль / Создать шлем + очки -> Goggle-Helmet Чтобы снять очки, просто раздавите шлем в дробилке (чары будут потеряны). Кузнечное дело Любой шлем + очки -> тот же шлем, но с прикрепленным NBT, поэтому он работает как защитный шлем. Чтобы снять его, объедините улучшенный шлем со средством для удаления модификаторов (созданным как ножницы, но из латуни) на столе кузнечного дела.

Если вы хотите создать свои собственные защитные шлемы для своего мода, посетите страницу github для получения дополнительной информации. Больше не поддерживается Minecraft 1.14, 1.16 и 1.18.
